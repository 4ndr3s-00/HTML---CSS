# Web Basics: HTML & CSS Reference

## Description
This project is a reference guide for HTML and CSS fundamentals. It includes structured examples of HTML tags and CSS properties commonly used in web development.

The goal is to provide a simple and reusable base to understand how web pages are built and styled.

---

## What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages.

HTML is based on **tags**, which define the elements of a page. For example:

- `<h1>` defines a heading
- `<p>` defines a paragraph
- `<a>` defines a link

Tags are used to organize content such as text, images, forms, and sections.

---

## What is CSS?

CSS (Cascading Style Sheets) is used to style and design web pages.

CSS works with **properties**, which define how elements look. For example:

- `color` changes text color
- `background` changes background color
- `margin` controls spacing
- `display` controls layout

CSS allows you to control layout, colors, spacing, and responsiveness.
